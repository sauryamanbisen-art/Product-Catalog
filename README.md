# Fully Responsive Amazon Clone

A beautifully designed, fully responsive clone of the Amazon homepage. 

## Features
- **Responsive Layout**: Adapts perfectly to Desktop, Tablet, and Mobile screens using modern CSS Flexbox and Grid.
- **Authentic Navigation**: Includes a comprehensive Amazon-style header with logo, location selector, categorised search bar, language options, sign-in, and an interactive cart.
- **Modern Styling**: Uses a visually appealing color palette (`#131921`, `#232f3e`, `#febd69`) and crisp styling to mimic the real Amazon experience.
- **Icons Integration**: Utilises FontAwesome via CDN for scalable, beautiful icons.
- **Micro-Animations**: Hover effects, smooth scaling on product images, and slide-in animations for a dynamic feel.
- **Optimized Assets**: Uses clearly named, locally stored images for quick loading.

## File Structure
- `Amazon.html`: The main structural layout of the web page.
- `Style/Amazon.css`: All the styling, responsive media queries, and animations.
- `public/image/`: The directory containing product images and the hero banner.

## Usage
Simply open `Amazon.html` in your web browser. Or, for a better development experience, run it on a local server using an extension like **Live Server** in VS Code.
